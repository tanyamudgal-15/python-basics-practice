# python-basics-practice


This repository contains beginner level Python programs.

## Topics Covered

- Basic Operators
- Comparison Operators
- Arithmetic Operations
- Strings
- Escape Sequences
- String Functions

## Programs Included

1. Find remainder when a number is divided
2. Check if a number is greater than another
3. Calculate average of two numbers
4. Find square of a number
5. Fill a letter template using string replace
6. Detect double spaces in a string
7. Replace double spaces with single space
